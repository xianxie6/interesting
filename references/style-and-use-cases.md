## 风格定位

把 `interesting` 理解为“忠于原图的视觉翻译系统”，而不是固定套用手、圆形或箭头的图标模板。

先识别原图为什么成立，再把它压缩成粗线、平涂、少颜色、强轮廓的新图。保留主体身份、空间关系、姿态和节奏；删除摄影质感、材质、光影与无关装饰。

## 核心风格特点

1. **原图骨架优先**
   - 先保留横竖方向、主体位置、大小比例、裁切、视线和遮挡关系。
   - 不为了套标准构图而改变原图的主判断。

2. **轮廓优先于细节**
   - 用一眼可辨的外轮廓表达人物、物体、树、建筑、器具或符号。
   - 只保留改变识别结果的内部线条。

3. **三色纯平涂**
   - 使用近黑 `#141413`、暖白 `#FAF9F6` 和一个固定背景色。
   - 禁止渐变、阴影、材质、透明高光和摄影式景深。

4. **粗黑手绘线**
   - 使用圆头、圆角、近似恒定的粗线。
   - 保留可控的自然抖动，避免完美矢量感和幼稚乱画感。

5. **低信息密度与留白**
   - 让一个主判断占据画面，其余区域保持安静。
   - 用三到五个重复单元建立节奏，例如枝条、栏杆、草叶、节点或平行线。

6. **人物与手均为可选**
   - 原图有人物或手时，保留其姿态、方向和作用。
   - 原图没有时默认不添加；不要把手当作风格水印。

7. **文字按信息价值处理**
   - 装饰性文字、界面标签和无关品牌默认删除。
   - 标题本身决定原图含义时，保留原文、层级和大致位置。

8. **两套配色必须同构**
   - 先完成一张构图，再只替换背景色。
   - 保持主体、文字、线条、比例和裁切完全一致。

## 场景路由

### 人物姿态与生活方式

用于人物被书本、报纸、器具或动作遮挡的照片，以及瑜伽、运动、阅读、工作等姿态。

- 保留人物姿势、重心、遮挡关系和标志性道具。
- 把服装褶皱、面部细节和材质压缩为少量轮廓。
- 参考 `01-relax-club-source-result.png` 和 `03-yoga-pose-result.png`。

### 风景、空间与海报

用于树、山、地平线、旷野、建筑和大留白海报。

- 保留主物体所在侧、地平线高度、远近比例和空白区域。
- 把树林、草地、云层或雪面压缩成大块形状与少量重复线。
- 标题是核心信息时保留，否则删除。
- 参考 `02-polished-snow-source-result.png` 和 `04-life-update-two-colorways.png`。

### 符号、品牌与动作关系

用于标志、图形符号与人物、手、器物之间存在明确关系的画面。

- 保留符号的基本拓扑、朝向和与动作主体的距离。
- 只在原图确有手或动作时保留手势。
- 避免把受保护标志改成相似但错误的结构。
- 参考 `05-symbol-and-hand-source-result.png`。

### 明信片、邮票与高密度装饰场景

用于原图身份依赖边框、植物、鸟、伞、浪线等多组重复元素的场景。

- 将整张场景视为一个语义核心，而不是机械限制为两个物体。
- 保留一个锚点主体、一个空间边界和必要的重复元素族。
- 控制为四到七类形状；避免恢复照片纹理。
- 参考 `06-beach-postcard-source-result.png`。

### 纯文字概念

用于没有原图、只有词语、段落、标题或抽象命题的请求。

- 使用 `object + action device + relationship` 生成隐喻。
- 先尝试位置、形变、连接和重复，再考虑手势。

## 适合与不适合

优先用于：

- 照片或海报的风格化重绘；
- 社交媒体封面、概念卡片和文章配图；
- 产品功能、抽象概念和品牌命题的视觉隐喻；
- 同一构图的双配色选择；
- 明信片、邮票或编辑插画式场景。

谨慎用于：

- 必须保留真人身份、产品材质或精确光照的写实任务；
- 数据图表、技术结构图和需要大量精确标签的画面；
- 长段正文排版；
- 多角色叙事且每个角色都必须精确还原的复杂场景。

## 最终检查

把结果与原图并排查看：

1. 两秒内是否认出同一主体和空间关系；
2. 是否出现原图没有的手、人物、道具或动作；
3. 是否只保留三种颜色；
4. 是否删除了摄影纹理、阴影和无关文字；
5. 双配色是否只改变背景；
6. 案例截图中的说明条、品牌字样和比较边框是否被误当成输出风格。
